AI for CODE

Let me give you a sneak peek at how AI can help with tasks involving code. Here I'll show you a couple of tasks - 'Translation' & 'Summarization'

In case you are not sure about what these tasks are, here is a quick intro :

    Translation - Translates code from one language to another. This can be useful if there is a need to translate legacy code from one language to another. Here I'm considering translation between Java and C# (i.e, Java->C# and C#->Java).
    Summarization - Summarizes code to English Language. This can be useful for generating docstrings automatically. Here I'm considering 6 languages - 'Go', 'Java', 'JavaScript', 'PHP', 'Python', 'Ruby'.

Note: This is just the tip of the iceberg. Currently there's a lot of research happening in this exciting area.
